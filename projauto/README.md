# Separador de Arquivos automatizado 📂

Esse projeto que fiz em  python pode separar "automaticamente" arquivos em uma pasta especifica, separada por tipos como (imagens, pdf's e documentos).

##  Como usar 🚀

01. Execute o script `sep.py`
02. Escolha no seu pc uma pasta ao abrir a janela
03. O programa irá criar subpastas de `imagens`, `pdfs` e `docs`, assim moverá automaticamente para as mesmas.

## Tecnologias usadas ✏️

- Python
- TKinter
- OS
